# Inner Thoughts Therapy Services

Website for Inner Thoughts Therapy Services — Wendy Mupaku, Clinical Social Worker & Psychotherapist, Grand Cayman.

## Tech Stack
- HTML5, Tailwind CSS 3, vanilla JavaScript
- Single-page responsive site

## Development
```bash
npm install
npm run dev    # Watch mode (auto-rebuild CSS on changes)
npm run build  # Production minified CSS
```

## Deployment
Connected to Netlify via GitHub. Pushes to `main` trigger automatic deploys.
